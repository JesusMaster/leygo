# Habilidad: Añadir Nodo Final al Supervisor del Agente
**Archivo:** `tool_1776193360.py`

## Descripción
Esta herramienta automatiza la modificación del archivo principal del núcleo del agente (`agent_core/main.py`) para integrar una condición de finalización explícita en el grafo de LangGraph. El script lee el contenido del archivo, inserta la importación de `END` desde la librería `langgraph.graph`, y luego añade un mapeo condicional `'FINISH': END` a la llamada del método `add_conditional_edges` del nodo `supervisor`. Esto asegura que el flujo de trabajo del agente pueda terminar de manera controlada cuando se cumpla la condición "FINISH". Finalmente, el script sobreescribe el archivo original con las modificaciones aplicadas.

## Requerimientos y Credenciales
Ninguno

## Parámetros Esperados (si se modifica)
*   `file_path`: Constante que define la ruta al archivo a modificar. Su valor predeterminado es `'agent_core/main.py'`.
*   `import_to_add`: Constante que contiene la línea de importación a añadir. Su valor predeterminado es `'from langgraph.graph import END'`.
*   `map_to_add`: Constante que define el mapeo de finalización que se insertará en la llamada a `add_conditional_edges`. Su valor predeterminado es `'FINISH': END`.
