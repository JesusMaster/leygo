# Habilidad: Calcular Distancia Geográfica
**Archivo:** `tool_1773751128.py`

## Descripción
Esta herramienta es un script de Python que calcula la distancia en kilómetros entre dos puntos geográficos en la superficie de la Tierra. Utiliza la fórmula de Haversine, que es un método preciso para calcular distancias sobre una esfera a partir de las latitudes y longitudes. El script está diseñado para ser ejecutado desde la línea de comandos, recibiendo las coordenadas de los dos puntos como cuatro argumentos numéricos: latitud 1, longitud 1, latitud 2 y longitud 2.

## Requerimientos y Credenciales
Ninguno

## Parámetros Esperados (si se modifica)
- `R`: Constante que representa el radio promedio de la Tierra en kilómetros. Su valor por defecto es `6371`. Modificar este valor podría usarse para calcular la distancia en otras unidades (por ejemplo, en millas, usando un valor aproximado de 3958.8).
