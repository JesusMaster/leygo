# Habilidad: Asegurar Condición de Finalización del Supervisor
**Archivo:** `tool_1776193585.py`

## Descripción
Esta herramienta modifica el archivo principal del agente (`agent_core/main.py`) para garantizar que el flujo de trabajo del supervisor, gestionado por LangGraph, tenga una condición de finalización explícita. Su función principal es doble: primero, se asegura de que la importación `from langgraph.graph import END` esté presente en el archivo, agregándola si es necesario. Segundo, localiza la definición de las rutas condicionales del nodo `supervisor` y añade el mapeo `'FINISH': END` a su diccionario de rutas. Esto es crucial para permitir que el agente concluya su ejecución de manera controlada cuando el supervisor determina que la tarea ha finalizado, evitando bucles infinitos o errores.

- `NODE_NAME`: El nombre del nodo cuyas aristas condicionales se van a modificar.
  - **Valor por defecto:** `'supervisor'`
- `MAPPING_TO_ADD`: El par clave-valor que se debe asegurar en el diccionario de rutas.
  - **Valor por defecto:** `'FINISH': END`
