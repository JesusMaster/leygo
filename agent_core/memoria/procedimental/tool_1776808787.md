# Habilidad: Eliminar Directorio Recursivo
**Archivo:** `tool_1776808787.py`

## Descripción
Esta herramienta elimina de forma permanente un directorio y todo su contenido (subdirectorios y archivos) de manera recursiva. Utiliza la función `shutil.rmtree` de la librería estándar de Python, lo que garantiza una eliminación completa y eficiente. Es ideal para limpiar estructuras de carpetas que ya no son necesarias.

**Advertencia:** La acción de esta herramienta es irreversible. Una vez que un directorio es eliminado, no puede ser recuperado por medios convencionales. Úsese con precaución.

## Requerimientos y Credenciales
Ninguno

## Parámetros Esperados (si se modifica)
*   `ruta_directorio`: Corresponde a la ruta (absoluta o relativa) del directorio que se desea eliminar. Este es el único argumento que la función principal `eliminar_directorio_recursivo` espera recibir.
